date_format - записывается формат даты, который будет поставлен в графе дата. Формат задается в следующих стилях
%%d.%%m.%%y отобразиться как 01.05.25
%%#d.%%#m.%%y отобразиться как 1.5.25
%%d.%%m.%%Y отобразиться как 01.05.2025
%%#d.%%#m.%%Y отобразиться как 1.5.2025

Можно так же использовать и другие форматы записи даты. Подробнее можно посмотреть функцию strftime https://docs.python.org/3/library/datetime.html#strftime-and-strptime-format-codes
Знак % необходимо дублировать

---------------------------------------

recopy - отвечает за перезапись существующих полей 1 - перезапишет заполненные поля (кроме шероховатости), 0 - заполненные поля не перезапишутся

---------------------------------------

first_used - отвечает за первичное применение, 1 - добавится первичное применение, 0 - не добавится
Рекомендуется: отключить если обзначение идет нестандартное, отличное от формата ГКЮШ.ХХХХ.ХХ.ХХ.ХХХ

---------------------------------------

rough_sign - 0 - Без указания типа отбработки; 1 - С удалением слоя материала; 2 - Без удаления слоя материала.
